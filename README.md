## Memory Usage

```bash
$ while sleep .2; do \
    free -m | grep ^Mem | awk '{print $2, $3, $4}'; \
  done | ./loadavg --title 'Memory Usage' --xlabel 'Time' --ylabel 'Size(MB)' \
	-l total -l used -l free
```
